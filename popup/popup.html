GitHub Fork Sync

How to connect:
  1. Go to GitHub Settings > Developer
  2. Click Generate new Token (classic)
  3. Check these 3 exact scopes:
    • repo (Full code access)
    • workflow (Fixes workflow sync errors)
    • admin:repo_hook (Fixes branch protection syncs)
  4. Paste the generated token below!
Stored securely on your browser. Never sent anywhere else.
